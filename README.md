# H-Bridge

## Components:
- 4N25 - 6 PIN DIP PHOTOTRANSISTOR PHOTOCOUPLER 
- AO3401A - P-Channel MOSFETs
- FDS6630A - N-Channel MOSFET
- 0805 SMD Resistors
- 2.54mm PinHeader Vertical Connectors 01x02
- 2.54mm PinHeader Vertical Connectors 01x01
